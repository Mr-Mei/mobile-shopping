# 介绍
本项目是利用业余时间开发，纯属练手，加深对各技术知识点的实操积累经验。基于 vue3 + axios 构建多页面开发的H5移动端电商网站，采用前后端分离，API接口调用使用mockjs模拟数据和第三方API接口（针对前端人员可以不涉及后端接口开发），H5界面分为：首页、分类搜索页、模糊搜索页，详情页、领券页、分享页。由于涉及微信页面，建议使用微信web开发者工具进行页面开发。
